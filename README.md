📊 Customer Shopping Behavior Analysis

🔍 Overview

This project focuses on analyzing customer shopping behavior to uncover insights that support data-driven business decisions. It covers the complete data analytics workflow—from data loading and cleaning to SQL analysis, dashboard creation, and reporting. The project demonstrates strong skills in Python, SQL, and data visualization tools.

📁 Dataset

The dataset contains customer transaction data, including:

Customer demographics (age, gender, location)
Purchase details (product category, amount, season)
Behavioral data (discount usage, purchase frequency, ratings, shipping type)

Size: ~3,900 records with multiple features

🛠️ Tools & Technologies
Python (Pandas, NumPy) – Data loading, cleaning, and EDA
SQL (PostgreSQL / MySQL / SQL Server) – Data analysis
Power BI – Interactive dashboard creation
Gamma – Presentation (PPT) generation
Excel (optional) – Data preview and validation

⚙️ Project Steps
1. Data Loading & Exploration
Loaded dataset using Pandas
Performed initial exploration using .info() and .describe()
2. Data Cleaning
Handled missing values
Standardized column names
Removed inconsistencies
3. Feature Engineering
Created new features like age groups
Derived purchase frequency metrics
4. SQL Analysis
Imported cleaned data into database
Performed queries for:
Revenue analysis
Customer segmentation
Product performance
Discount behavior
5. Data Visualization
Built an interactive Power BI dashboard
Added filters for better user interaction
6. Reporting & Presentation
Created a detailed report of insights
Designed a professional presentation using Gamma

📊 Dashboard

The Power BI dashboard provides:

Revenue insights by customer segments
Product performance analysis
Customer behavior trends
Interactive filters (category, gender, subscription, etc.)

📈 Key Results
Identified high-value customer segments
Found top-performing products and categories
Analyzed impact of discounts on sales
Discovered opportunities to improve subscription conversion
Provided actionable business recommendations

▶️ How to Run

Clone the repository:

git clone https://github.com/Karan5787/customer_behavior_analysis

Install required Python libraries:

pip install pandas numpy sqlalchemy psycopg2
Run the Python script/notebook for data cleaning and preprocessing
Load cleaned data into your SQL database (PostgreSQL/MySQL/SQL Server)
Execute SQL queries from the /sql folder
Open the Power BI file (.pbix) to view the dashboard
